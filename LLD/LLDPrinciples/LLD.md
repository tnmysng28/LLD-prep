- DRY: Dont repeat Yourself
    -- template design pattern
    -- repeating yourself may increase chance of errors
    -- Code deduplication
    -- Improves maintainability and enhances dev productivity


- KISS: Keep it Simple Stupid
    -- Keep things simple and straightforward
    -- Maximize clarity and reduce load to understand the code
    -- No over engineering
    -- Makes code easy to understand, debug and maintain. 
    -- Managable.
